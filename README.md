# Random 3d Engine (R3E)
This is my little project that I made when I was bored.
It's built on love2d v11.5
It features polygon rendering, importing models from STL (ASCII) format and bad lighting.

==CONTROLS==\n
W, A, S, D - Move
Space, Shift - Move up/down
kp+, kp- - Change speed
Arrows - Rotate world
